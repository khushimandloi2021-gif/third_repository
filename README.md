# Chai Wala Billing System ☕

A fun terminal-based billing app for a chai shop!

## Features
- Choose between Chai or Coffee
- Add extra sugar, rusk, bread
- Auto bill calculation

## Menu
- Chai = Rs.20
- Coffee = Rs.30
- Extra Sugar = Rs.2
- Rusk = Rs.5
- Bread = Rs.5

## Concepts used
- if-elif-else conditions
- User input handling
- Bill calculation logic
